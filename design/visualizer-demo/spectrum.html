SPECTRUM · 频谱

DESIGN NOTES · 设计说明

  • 迹线语言与 EQ 页同源：主曲线 = 印章红 2px + 8% 红晕填充，网格 = line 色 0.5px，主频率线（100 / 1k / 10k）加深——用户在 EQ 页养成的阅读习惯直接迁移。
  • 峰值保持：墨灰虚线以 3.5 dB/s 缓落，展示乐句轮廓；虚线 vs 实线的层级不依赖颜色。
  • 坐标：横轴对数 20Hz–20kHz（96 个 1/12 倍频程量级频点），纵轴 0 ~ -90 dB，右缘每 20dB 标数。
  • 游标读数：悬停出现墨色点划竖线 + 曲线上的印章红取样点，频率 / dB 显示在面板头右侧固定读数位——仪表习惯，不用浮动气泡遮挡曲线。
  • 键盘可达：本页画布可聚焦，◀ ▶ 移动游标、Esc 清除——不依赖悬停的读数途径。
  • 窄面板降级：宽度不足 520px 时频率标签自动收敛为 100 / 1k / 10k 三档，避免挤压。
  • 实装建议：seraph-visualizer 已做 FFT，取幅度谱后按 1/12 oct 聚合 + attack/release 平滑（快攻 ~40ms / 慢放 ~140ms）即为此曲线；每帧一个 Float32Array(96)。
  • 填充只用 8% 透明红——曲线是主角，避免重底色喧宾夺主。
