DESIGN NOTES · 设计说明

  • 双层条：实墨填充 = RMS（听感能量），浅棕填充 = 瞬时峰值，两层同槽叠放；专业表习惯 + 纸面「深浅墨」的档案质感。
  • 峰值保持线：横杠保持 1.4s 后以 14 dB/s 回落；进入红区（> -6 dBFS）时保持线由档案棕转印章红。
  • 红区（0 ~ -6 dBFS）用印章红斜阴影线铺底——不靠纯色块，色弱用户也能靠纹理识别危险区。
  • 刻度：0 / -6 / -12 / -24 / -40 / -60 主刻度带数字，-3 / -9 / -18 / -30 / -50 为细分线；dB 线性布尺。
  • CLIP 印章：任一声道峰值触及 -1.2 dBFS 即盖章，保持 3s 自动解除；数字读数同步变红（双通道语义）。
  • 数字读数置于条形之上：PEAK 显示保持值、RMS 显示积分值，Courier 等宽避免跳动。
  • 实装建议：复用 seraph-audio 输出回调里的样本块，RMS 用 300ms 滑窗，峰值直接取块内最大绝对值；每帧只需 4 个标量。
  • 条宽自适应面板宽度（22–52px 之间），窄窗不挤压、宽窗不虚胖。
